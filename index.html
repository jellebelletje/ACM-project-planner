How to connect a new project to Google Sheets

  1. Copy the Google Sheet data sheet: Open the template ACM planner data sheet, go to File → Make a copy. This copies all tabs and the attached Apps Script.
  2. Deploy as a Web App: Click Deploy → New deployment. Set Execute as to Me and Who has access to Anyone. Click Deploy.
  3. Copy the Web App URL that appears after deployment (it looks like https://script.google.com/macros/s/.../exec).
  4. Open the project in the dashboard, go to Settings (gear icon), and paste the URL into the API URL field. Click Save.
  5. Seed the data (optional): Open seed.html in your browser, enter the same API URL, and click Seed All to populate the sheet with the standard ACM activity template.

Each project needs its own Google Sheet and its own Apps Script deployment. The dashboard will work offline with cached data even if the sheet is unreachable.
